# svad-ba

Промо-сайт свадебной медиа-кампании Артема и Евы.  
Проект сделан как яркий одностраничный лендинг с кастомной вёрсткой, графикой и адаптацией под мобильные устройства.

## Что внутри

- `index.html` - основная страница сайта
- `colors.css` - палитра и вспомогательные стили
- `assets/` - фотографии, стикеры и декоративная графика
- `capture-fullshots.mjs` - локальный скрипт для снятия полноразмерных скриншотов через Playwright

## Локальный запуск

Сайт статический, поэтому его можно открыть прямо через `index.html`.

Если нужен локальный сервер, можно запустить, например:

```bash
npx serve .
```

Или через Vite preview / любой другой простой HTTP server, если он у тебя уже есть.

## Зависимости

В репозитории используются:

- `playwright`
- `puppeteer`

Они нужны не для работы самого сайта в браузере, а для вспомогательных задач вроде автоматизированных проверок и скриншотов.

## Скриншоты

В репозитории не хранятся локальные QA-скриншоты и служебные браузерные файлы:

- `node_modules/`
- `.playwright-browsers/`
- `qa-*.png`
- `screenshot-*.png`

Это исключено через `.gitignore`.
